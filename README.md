# 勘误

## 关于Angular2的勘误
Angular开发组原本宣称只支持 IE11+，现在把兼容性起点设置为IE9了，这意味着等它正式发布的时候，将可以完全替代现在的Angular 1.3的应用场景。

不过，如果需要支持IE8，那么仍然只能使用Angular 1.2。
